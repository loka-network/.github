## ğŸ¯ Overview

**Loka** is the financial operating system for the agentic economy. As artificial intelligence becomes a primary economic actor, it requires a new monetary layerâ€”one that operates at machine speed, is programmatically verifiable, and settles value based on compute, not just balances.

### The Core Problem: Capital Velocity Bottleneck

The next $10 trillion of GDP will be generated by machines, but money still moves at human speed. Existing financial rails, including traditional stablecoins, were not designed for the high-frequency, low-friction, and continuous nature of machine-to-machine transactions. This creates a fundamental **capital velocity problem**, where AI's potential is bottlenecked by antiquated financial infrastructure.

Loka solves this by creating a closed-loop system where compute consumption is directly tied to monetary settlement, enabling a new generation of autonomous economic activity.

### Architecture Layer Details

1.  **Settlement & Channel Layer**: A dual-ledger system featuring **Loka Chain**, a high-performance EVM L1 optimized for stablecoin settlement (30k+ TPS, <1s finality), and **Setu RIM**, an off-chain payment channel network for instant, zero-fee micropayments.
2.  **Payment & Risk Layer**: A cluster of microservices running in **AWS Nitro Enclaves (TEE)**. This layer handles sensitive operations like key management, risk assessment, and payment orchestration, communicating asynchronously via the **Nostr** protocol.
3.  **Payment Experience Layer**: Delivered via **Agent SDKs (SKILLs)**, this layer embeds cryptographic capabilities directly into AI agent frameworks, providing tools for ZK-powered privacy, session keys, and automated payments.
